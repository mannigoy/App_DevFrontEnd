#  Campus Marketplace

> A centralized campus marketplace app that allows students to easily browse available products, check stocks, and place orders — eliminating long lines and lack of visibility on campus goods and services.

---




## Overview

**Campus Marketplace** is a full-stack web application built with **React** (frontend) and **Spring Boot** (backend). It serves as a centralized platform for the campus community to browse, manage, and purchase products and services without the hassle of physical queues or uncertainty about stock availability.

---

## Pain Point

> Students experience inconvenience and wasted time due to long lines, unclear stock availability, and the lack of a centralized platform to easily find and purchase campus products and services.

## Features (MoSCoW)

###  Must-Have
These are the core features required for the app to function:

1. **Product Listing** — Display all available products to users
2. **Stock Management** — Allow sellers to manage and update their inventory
3. **Search and Category Filter** — Enable users to search for products and filter by category
4. **Order System** — Allow buyers to place orders from sellers
5. **Payment** — Enable payment processing for ordered products

---

###  Should-Have
Important features that add significant value:

1. **Pick-up Schedule** — Let buyers choose a time and location for product pick-up
2. **Notifications** — Notify sellers of incoming orders; notify buyers of order status updates
3. **Order History** — Allow buyers to view their past orders
4. **Product Details** — Sellers can edit product name, price, and description; buyers can view full product details
5. **Ratings & Reviews** — Buyers can leave reviews on products

---

###  Nice-to-Have
Features that enhance the experience but are not critical:

1. **Dashboard** — Sellers see sales statistics, order notifications, and stock levels at a glance
2. **AI Recommendations** — An AI engine that recommends products to buyers based on purchase history

## Getting Started

### Prerequisites
- Node.js & npm
- Java 17+
- Maven
- Git

### Frontend (React)

```bash
# Clone the repository
git clone https://github.com/mannigoy/Campus_Marketplace.git

# Navigate to the frontend directory
cd Campus_Marketplace/frontend

# Install dependencies
npm install

# Start the development server
npm start
```

### Backend (Spring Boot)

```bash
# Navigate to the backend directory
cd Campus_Marketplace/backend

# Build and run using Maven
./mvnw spring-boot:run
```
